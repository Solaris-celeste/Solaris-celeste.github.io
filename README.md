# My Personal Website

This is the source code for my personal website, built with [Astro](https://astro.build) and deployed via GitHub Actions.

##  Project Structure

- src/pages: Website pages
- src/content/blog: Blog posts
- src/components: UI components
- src/layouts: Page layouts
- src/styles: Global styles (Tailwind CSS)

##  Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| \
pm install\             | Installs dependencies                            |
| \
pm run dev\             | Starts local dev server at \localhost:4321\      |
| \
pm run build\           | Build your production site to \./dist/\          |
| \
pm run preview\         | Preview your build locally, before deploying     |

##  Want to learn more?

Feel free to check [Astro documentation](https://docs.astro.build) or [Tailwind CSS documentation](https://tailwindcss.com/docs).
